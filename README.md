## Описание

Данная программа была разработанна в качестве frontend для проекта "credit-service-backend"

## Требования

- Node - 18v

## Установка зависимостей

```bash
$ npm install
```

## Запуск проекта

```bash
$ npm run build
```

```bash
$ npm run preview
```
